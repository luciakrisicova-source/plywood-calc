Kalkulačka dosiek - PWA

Ako spustiť:
1. Otvor priečinok vo Visual Studio Code.
2. Nainštaluj rozšírenie Live Server.
3. Pravým klikom otvor index.html cez "Open with Live Server".
4. V mobile otvor rovnakú adresu cez Wi-Fi sieť alebo projekt nahraj na hosting.
5. V mobile daj "Pridať na plochu".

Súbory:
- index.html = formulár a rozloženie
- style.css = vzhľad
- app.js = výpočty
- manifest.json = nastavenie PWA aplikácie
- service-worker.js = offline režim
- icon.svg = ikona aplikácie
